## CabinBuilder 门窗不可见 bug — 两轮修复

### 第一轮修复 (Y 坐标 + 侧面)
- `_addInstancedBlocks` 的 Y 偏移 `y*BLOCK+BLOCK*0.5` 与门/窗不一致
- 窗户在侧面墙 (x=0/x=4)，摄像机看不到
- Raycaster 改为检测 `doorGroup.children`，WelcomeApp 添加 doorGroup 属性

### 第二轮修复 (正面方向彻底搞反!)
**根因**: 摄像机在 z=+18 看向 z=0，所以面向摄像机的"正面"是 z=D-1 (bz=+1)，而非 z=0 (bz=-2)。门和窗全放在了 z=0 即最远的背面。
**修复**: 
- 门洞/窗户跳过条件从 `z===0` 改为 `z===D-1`（FRONT_Z）
- frontWorldZ = (D-1 - halfD)*BLOCK = +1
- 门突出方向从 Z- 改为 Z+（朝摄像机）: `doorFrontZ = frontWorldZ + 0.6`
- 开门动画方向从 -π/2 改为 +π/2（向外开朝摄像机）
- 门把手凸出方向改为 Z+
- 门后光改为 Z- 方向
